# OPNsense Setup
Documentation for my OPNsense configuration and home network build.  
This project demonstrates hands‑on experience with firewall deployment, network design, access point integration, and troubleshooting in a real environment.

## Current State
- Base install complete on bare‑metal hardware
- LAN/WAN interfaces configured
- Access point connected and operating in bridge mode
- Static IP assignments for core devices
- Initial firewall ruleset in place

## Skills Demonstrated
- Deploying and configuring OPNsense on dedicated hardware
- Interface assignment, routing validation, and DHCP behavior analysis
- Integrating a WiFi 6 access point with proper IP planning
- Troubleshooting and reinstalling after a corrupted initial deployment
- Documenting network behavior and configuration decisions
